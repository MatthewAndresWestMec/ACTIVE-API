## City Weather

Enter the exact name of a city (state is optional) to find the following: Humidity, Climate, Temperature, Wind, Rain, HI/LOW, UV, Date
Link: https://www.weatherapi.com/

## Country Data

Enter the exact name of a country to find the following: Language, GDP, Region, Currency, Population, Flag, Coat of Arms
Link: https://geography.mmplabs.com/

To run this API, follow these steps: 1. Ensure you have Node.js installed. 2. Install the required dependencies using `npm install`. 3. Create a .env file in the root of the project with three variables: `PORT`, `WEATHER`, and `GEOGRAPHY` 4. Set `PORT` to your preferred port for this API. 5. Set both `WEATHER` and `GEOGRAPHY` to their respective API keys. 6. Run the application using `npm start` or your preferred command.
